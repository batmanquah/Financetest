# My Finance App

This is a simple installable personal finance web app.

## Files
- index.html: main app
- manifest.json: app install settings
- service-worker.js: offline support
- icon.svg: app icon

## iPhone use
1. Upload these files to a simple web host, GitHub Pages, Netlify, or Vercel.
2. Open the website in Safari.
3. Tap Share.
4. Tap Add to Home Screen.

Your data is stored locally on your phone/browser.
